# Algoritmos em C

Repositório contendo implementações desenvolvidas em linguagem C voltadas para lógica de programação, manipulação numérica, estruturas condicionais, estruturas de repetição e resolução de problemas algorítmicos.

Os códigos foram organizados de forma separada para facilitar a visualização da lógica utilizada em cada problema, abordando cálculos matemáticos, manipulação de datas, sequências numéricas e análise de dados.

O objetivo deste portfólio é registrar estudos e aplicações práticas utilizando a linguagem C, mantendo os arquivos organizados e documentados individualmente.

---

## Conteúdo

O repositório contém programas relacionados a:

- Manipulação de datas e horários
- Cálculos envolvendo anos bissextos
- Sequências numéricas modulares
- Estruturas de repetição
- Estruturas condicionais
- Operações matemáticas
- Análise de sequências
- Sistemas simples de pontuação
- Lógica algorítmica

---

## Organização

Cada arquivo possui uma implementação específica, a pasta possui um README com explicações mais detalhadas sobre funcionamento, lógica utilizada e objetivo do programa.
