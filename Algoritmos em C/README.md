# Arquivos

## dia_do_ano_1.c

Programa que recebe uma data no formato `dd/mm/aaaa` e calcula o dia correspondente dentro do ano.

O código realiza o tratamento de anos bissextos e utiliza estruturas condicionais e `switch case` para somar corretamente a quantidade de dias dos meses anteriores.

---

## dia_do_mes_em_porcentagem_2.c

Programa que recebe uma data, horário e um percentual, calculando a data e horário equivalentes ao percentual informado dentro do ano.

O algoritmo converte a data para minutos totais, aplica o percentual informado e reconstrói a nova data considerando anos bissextos.

---

## lista_de_1_a_9_3A.c

Implementação de uma sequência numérica modular utilizando dois valores iniciais.

A sequência é gerada a partir da soma dos dois últimos números módulo 10, acumulando os valores até atingir um limite informado pelo usuário.

---

## lista_de_soma_3B.c

Programa que gera uma sequência modular e calcula a soma total dos elementos até o reinício do ciclo original.

O algoritmo identifica automaticamente quando a sequência retorna aos valores iniciais informados.

---

## sequencia_digitos_3C.c

Programa que analisa sequências modulares geradas a partir de pares numéricos de 1 a 9.

O código identifica quais pares produzem as maiores e menores sequências antes da repetição do ciclo.

---

## verifica_valor_na_sequencia_3D.c

Programa responsável por verificar se um determinado valor aparece dentro de uma sequência modular gerada a partir de dois números iniciais.

A verificação ocorre durante toda a geração da sequência até o retorno ao ciclo original.

---

## pontuacao_surfistas_4.c

Sistema de pontuação entre dois surfistas.

O programa recebe notas, calcula automaticamente as duas maiores pontuações de cada competidor, identifica o líder atual e informa a nota necessária para ultrapassagem.
